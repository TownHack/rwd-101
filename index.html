Some Statistics

  1. Desktop usage has dropped from about 82% to 62%
  2. Mobile usage has raised from about 13% to 31%
  3. Tablet/Console usage has had a slight increase

What does this mean?

- Something needs to change - We can't keep trying to develop for each possible outcome independently
